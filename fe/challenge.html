Galdus Challenges

These are 7 galdus challenges:


Challenge 1:

  • First create an HTML file.

  • Then add an h1 and a few paragraphs.

  • After that launch the server and view the galdus page on your browser.


Challenge 2:

  • Create a JS file and connect it to the HTML page.

  • Delete p tags that contain a word of your choice, without touching the others.

Challenge 3:

  • Modify the colour and add a label.

Challenge 4:

  • Add a button.

  • Manage the click of the button by writing a message on the console.

Challenge 5:

  • Change the order in which posts are displayed from the most recent one, in order of insertion.

Challenge 6:

  • Add a new field in DB: author (string type, not null).

Challenge 7:

  • Manage the new author field in the BE by modifying the two functions /api/post/add and /api/posts.

  • Display the new author field in the FE and manage the input.

Super Challenge:

  • BE: Add the delete function of a post: /api/post/delete/:idPost

  • FE: Add the delete button of the post (next to the title), call the BE delete function and update the list of posts.
